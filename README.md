# suicanfc kd

このAndroidアプリは、日本で広く利用されている交通系NFCカード **Suica** 、**Pasmo** 、**ICOCA** 、**manaca**などのデータを読み取り、表示するために作られています。

このアプリでは、カードから取得した **データを確認** することができます。
プライバシー重視でインターネットに接続する権限はないためテレメトリなどはありません。

## 最新APKバージョン

最新APKバージョンは[リリース](https://github.com/Kdroidwin/SuicaNFCReader-modern-source/releases)からダウンロードできます。



## 機能

- SuicaおよびPasmoカードなどのデータを読み取り、表示します。
- データを確認しやすいインターフェースを提供します。
- カードデータは安全かつプライベートに扱われます。
- テーマ変更
- 履歴保存
- 複数のカードに対応
- ICOCA判定
- カード名手動編集
- タイトル名手動編集
- 駅、鉄道会社名手動編集（手動編集した部分は更新しないように）
- CSV JSON Notion 形式でインポート　エクスポート
- 統計機能
- テーマ変更・アクセント色の変更
- メモ機能
- タグ機能
- 履歴検索　フィルタ
- ウィジェット対応
