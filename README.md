# Suica NFC Reader by Kdroidwin

このAndroidアプリは、日本で広く利用されている交通系NFCカード **Suica** および **Pasmo** **ICOCA** **manaca**のデータを読み取り、表示するために作られています。

このアプリでは、カードから取得した **データを確認** することができます。

## 最新APKバージョン

最新APKバージョンはリリースからダウンロードできます。


## 機能

- SuicaおよびPasmoカードなどのデータを読み取り、表示します。
- データを確認しやすいインターフェースを提供します。
- カードデータは安全かつプライベートに扱われます。
- テーマ変更
- 履歴保存
- 複数のカードに対応
